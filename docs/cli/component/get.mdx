# arcctl component get

Get detailed information about a deployed component.

## Synopsis

```bash
arcctl component get <name> -e <environment> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Component name |

## Options

| Option | Description |
|--------|-------------|
| `-e, --environment <name>` | Target environment (required) |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |
| `--output <format>` | Output format: `table`, `json`, `yaml` |

## Examples

```bash
# Get component details
arcctl component get web-app -e staging

# JSON output
arcctl component get web-app -e staging --output json
```
