# arcctl datacenter destroy

Destroy a datacenter and all its resources.

<Warning>
This will destroy all environments in the datacenter. Use with caution.
</Warning>

<Note>
Use `arcctl dc` as shorthand for `arcctl datacenter`.
</Note>

## Synopsis

```bash
arcctl datacenter destroy <name> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Datacenter name |

## Options

| Option | Description |
|--------|-------------|
| `--auto-approve` | Skip confirmation prompt |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |

## Examples

```bash
# Destroy a datacenter
arcctl dc destroy aws-staging

# Destroy with auto-approval
arcctl dc destroy aws-staging --auto-approve
```
