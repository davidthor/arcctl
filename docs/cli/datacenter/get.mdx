# arcctl datacenter get

Get detailed information about a datacenter.

<Note>
Use `arcctl dc` as shorthand for `arcctl datacenter`.
</Note>

## Synopsis

```bash
arcctl datacenter get <name> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Datacenter name |

## Options

| Option | Description |
|--------|-------------|
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |
| `--output <format>` | Output format: `table`, `json`, `yaml` |

## Examples

```bash
# Get datacenter details
arcctl dc get aws-production

# JSON output
arcctl dc get aws-production --output json
```
