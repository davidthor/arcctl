# arcctl datacenter list

List all deployed datacenters.

<Note>
Use `arcctl dc` as shorthand for `arcctl datacenter`.
</Note>

## Synopsis

```bash
arcctl datacenter list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |
| `--output <format>` | Output format: `table`, `json`, `yaml` |

## Examples

```bash
# List datacenters
arcctl dc list

# JSON output
arcctl dc list --output json
```

## Output

```
$ arcctl dc list

NAME              SOURCE                                    ENVIRONMENTS
aws-production    ghcr.io/myorg/aws-datacenter:v1.0.0      staging, production
aws-staging       ghcr.io/myorg/aws-datacenter:v0.9.0      dev, preview-*
local-docker      ghcr.io/myorg/local-datacenter:latest    dev
```
