# arcctl datacenter push

Push a datacenter artifact and all its module artifacts to an OCI registry.

<Note>
Use `arcctl dc` as shorthand for `arcctl datacenter`.
</Note>

## Synopsis

```bash
arcctl datacenter push <repo:tag> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<repo:tag>` | Datacenter artifact reference to push |

## Options

| Option | Description |
|--------|-------------|
| `--yes, -y` | Non-interactive mode |

## Examples

```bash
# Push to registry
arcctl dc push ghcr.io/myorg/aws-datacenter:v1.0.0

# CI/CD push
arcctl dc push ghcr.io/myorg/aws-datacenter:$CI_COMMIT_SHA --yes
```
