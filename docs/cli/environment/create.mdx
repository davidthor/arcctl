# arcctl environment create

Create a new environment.

<Note>
Use `arcctl env` as shorthand for `arcctl environment`.
</Note>

## Synopsis

```bash
arcctl environment create <name> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Environment name |

## Options

| Option | Description |
|--------|-------------|
| `-d, --datacenter <name>` | Target datacenter (required for new environments) |
| `--if-not-exists` | Don't error if environment already exists |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |

## Examples

```bash
# Create an environment
arcctl env create staging --datacenter aws-production

# Create if not exists (idempotent)
arcctl env create staging --datacenter aws-production --if-not-exists

# Using alias
arcctl env create preview-123 -d aws-staging
```

## Output

```
$ arcctl env create staging --datacenter aws-production

Environment: staging
Datacenter:  aws-production

[create] Creating environment "staging"...
[success] Environment created successfully
```
