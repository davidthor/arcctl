# arcctl environment get

Get detailed information about an environment.

<Note>
Use `arcctl env` as shorthand for `arcctl environment`.
</Note>

## Synopsis

```bash
arcctl environment get <name> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<name>` | Environment name |

## Options

| Option | Description |
|--------|-------------|
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |
| `--output <format>` | Output format: `table`, `json`, `yaml` |

## Examples

```bash
# Get environment details
arcctl env get staging

# JSON output
arcctl env get staging --output json
```
