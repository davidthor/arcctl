# arcctl environment list

List all environments.

<Note>
Use `arcctl env` as shorthand for `arcctl environment`.
</Note>

## Synopsis

```bash
arcctl environment list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `-d, --datacenter <name>` | Filter by datacenter |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |
| `--output <format>` | Output format: `table`, `json`, `yaml` |

## Examples

```bash
# List all environments
arcctl env list

# Filter by datacenter
arcctl env list --datacenter aws-production

# JSON output
arcctl env list --output json
```

## Output

```
$ arcctl env list

NAME           DATACENTER         COMPONENTS   CREATED
staging        aws-production     4            2026-01-20
production     aws-production     4            2026-01-25
dev            local-docker       2            2026-01-28
preview-123    aws-staging        3            2026-01-29
```
