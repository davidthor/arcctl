# cldctl list datacenter

List all deployed datacenters.

<Note>
Use `cldctl ls dc` or `cldctl list dc` as shorthand.
</Note>

## Synopsis

```bash
cldctl list datacenter [options]
```

## Options

| Option | Description |
|--------|-------------|
| `-o, --output <format>` | Output format: `table`, `json`, `yaml` |
| `--backend <type>` | State backend type |
| `--backend-config <key=value>` | Backend configuration |

## Examples

```bash
# List datacenters
cldctl list datacenter

# Using alias
cldctl ls dc

# Output as JSON
cldctl list datacenter -o json

# Output as YAML
cldctl list datacenter -o yaml
```

## Output

```
$ cldctl list datacenter

NAME              SOURCE                                        ENVIRONMENTS
aws-production    ghcr.io/myorg/aws-dc:v1.0.0                  production, staging
aws-dev           ghcr.io/myorg/aws-dc:v1.2.0                  dev, preview-123
local             local                                         local-dev
```
