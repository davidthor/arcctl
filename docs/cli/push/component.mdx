# cldctl push component

Push a component artifact and all its child artifacts to an OCI registry.

<Note>
Use `cldctl push comp` as shorthand for `cldctl push component`.
</Note>

## Synopsis

```bash
cldctl push component <repo:tag> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<repo:tag>` | Component artifact reference |

## Options

| Option | Description |
|--------|-------------|
| `-y, --yes` | Non-interactive mode |

## Examples

```bash
# Push a component
cldctl push component ghcr.io/myorg/myapp:v1.0.0

# Using alias
cldctl push comp ghcr.io/myorg/myapp:v1.0.0

# Non-interactive
cldctl push component myapp:latest -y
```

## Output

```
$ cldctl push component ghcr.io/myorg/myapp:v1.0.0

Pushing component artifact: ghcr.io/myorg/myapp:v1.0.0

Proceed with push? [Y/n]:
[push] Pushing ghcr.io/myorg/myapp:v1.0.0...
[success] Pushed ghcr.io/myorg/myapp:v1.0.0
```
