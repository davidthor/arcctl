# cldctl push datacenter

Push a datacenter artifact and all its module artifacts to an OCI registry.

<Note>
Use `cldctl push dc` as shorthand for `cldctl push datacenter`.
</Note>

## Synopsis

```bash
cldctl push datacenter <repo:tag> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<repo:tag>` | Datacenter artifact reference |

## Options

| Option | Description |
|--------|-------------|
| `-y, --yes` | Non-interactive mode |

## Examples

```bash
# Push a datacenter
cldctl push datacenter ghcr.io/myorg/dc:v1.0.0

# Using alias
cldctl push dc ghcr.io/myorg/dc:v1.0.0

# Non-interactive
cldctl push datacenter my-dc:latest -y
```

## Output

```
$ cldctl push datacenter ghcr.io/myorg/dc:v1.0.0

Pushing datacenter artifact: ghcr.io/myorg/dc:v1.0.0

Proceed with push? [Y/n]:
[push] Pushing ghcr.io/myorg/dc:v1.0.0...
[success] Pushed ghcr.io/myorg/dc:v1.0.0
```
